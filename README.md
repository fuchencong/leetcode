## 0009

反转数字：

## 0154

可重复数字旋转之后，寻找最小数

* 旋转之后，数组分为两部分，左半部分的数均大于等于右半边的数
* 使用 mid 和右边边界进行比较，如果 mid 大，则 mid 肯定是在左半部分，则分隔点在 mid+1 之后
* 如果 mid 小，则 mid 肯定在右半边部分，则分隔点 mid 之前（可以等于 mid）
* 如果相等，则直接缩小右边界（-1）
